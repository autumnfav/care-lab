# CARE Lab Website

이 레포는 정적 랩 웹사이트다 (단일 `index.html`).

## 규칙

- 콘텐츠(텍스트, 출판물, 명단) 수정 요청 시 **HTML 구조·CSS·디자인은 절대 변경하지 말 것.**
- 디자인 변경은 명시적 요청이 있을 때만.
- 자주 수정되는 구역은 `index.html` 안에 `<!-- ✏️ EDIT: ... START/END -->` 주석 마커로 표시되어 있다:
  - `RECRUITING (home hero banner)` — 홈 히어로 모집 배너
  - `PUBLICATIONS LIST` — 출판물 목록
  - `PEOPLE` — 구성원 명단
  - `RECRUITING (News item)` — News 페이지 모집 공지
  - `RECRUITING (Join page note)` — Join 페이지 모집 문구
