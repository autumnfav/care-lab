# CARE Lab — Child, Adolescent, Resilience, and Ecology Lab

Website for the CARE Lab at Texas Tech University, directed by Dr. Wonjung Oh. The whole site is one file (`index.html`) — no build step. Pushing to `main` publishes it automatically via GitHub Pages.

**Editing:** just ask Claude Code in plain language — both content (publications, people, news) and design (colors, layout, typography) can be changed. Don't worry about breaking things: every change is saved in git history, so anything can be restored at any time.
